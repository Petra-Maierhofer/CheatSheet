Sample Questions

Word Wide Web General

1. What are the most essential technologies that made the World Wide Web such a success?

HTML, HTTP, URL, Web Browsers, Web Servers, TCP/IP, Hyperlinks, CSS, JavaScript, Search Engines

2. How are resources (html pages, images, etc) addressed in the web?

Resources are addressed using URLs (Uniform Resource Locators), which provide a unique address for each resource on the web.

3. What are the general components of a URL?

The general components of a URL are: scheme (protocol), host (domain), port, path, query string, and fragment.

Web Server

1. What is a web server?

A web server is software and hardware that stores, processes, and delivers web pages to clients using HTTP.

2. What does a web server?

A web server handles requests from clients, retrieves requested resources, and sends them back to the clients over the internet.

3. How can content be stored on a web server?

Content can be stored as HTML files, images, videos, CSS, JavaScript files, or any other static or dynamic content.

4. How is content in webpages described?

Content in web pages is described using HTML, which provides structure, and CSS, which provides visual styling.

Web Client

1. What is a web client?

A web client is software, typically a web browser, that sends requests to a web server and displays the retrieved content.

2. What does a web client?

A web client sends requests to web servers, retrieves content, and renders that content for users to view and interact with.

3. Web Client and Web Browser is the same

A web browser is a type of web client used to access and display web pages. Not all web clients are browsers, but all browsers are web clients.

Hello World

1. What is the html head element?

The HTML head element contains meta-information about the document, such as the title, character set, and links to stylesheets and scripts.

2. What does <!DOCTYPE html> mean?

<!DOCTYPE html> declares the document type and version, specifying that the document is an HTML5 document.

3. What are the two top most elements within the html element?

The two topmost elements within the HTML element are <head> and <body>.

4. How to emphasize a text and how is it displayed in the browser?

To emphasize text, use the <em> tag. It is typically displayed in italics in the browser.

5. How to mark up important text and how is it displayed in the browser?

To mark up important text, use the <strong> tag. It is typically displayed in bold in the browser.
